Major Introduction

My major, Computer Science, covers core knowledge and skills in a wide range of areas such as computer systems, programming languages, software engineering, data management and web development. Through theoretical studies and practical exercises, I will systematically master computer hardware and software design, programming and algorithm design, software development methodologies, database management and web application development, and have the ability to solve complex problems and develop efficient applications.

Course 1: Python Programming Foundation

This course provides a basic introduction to computer programming, teaching fundamental programming concepts such as variables, conditional statements, and loops from the ground up. Students will learn how to write and debug Python programs, understand functions, recursion, and modular programming, and master the use of floating-point numbers and scientific computing libraries to build a solid foundation for subsequent programming courses.

Course 2: Object-Oriented Programming

This course provides an in-depth introduction to the concepts and practices of object-oriented programming, utilising Python as the programming language. The course will provide students with an understanding of data structures and algorithms, including stacks, queues, chain tables, trees, maps, dictionaries, and hash tables. Additionally, students will gain insight into software engineering practices, such as modular programming, object-oriented analysis, design, and testing. This will enhance their ability to develop complex software systems.

Course 3: Computer Architecture

This course provides an introduction to the fundamental principles underlying the operation of computers, encompassing topics such as number systems (including binary and hexadecimal), logic gates, Boolean algebra, and arithmetic circuits. The course will examine the fundamentals of computer architecture, including assembly language, machine language, and the microarchitecture of processor design (single-cycle, multi-cycle, and pipelined processors) and memory systems (caches and virtual memory). The objective is to provide students with a comprehensive understanding of the core construction and operation of computer systems.

Course 4: Introduction to Software Engineering

This course introduces the fundamental principles and practices of software engineering, including the software development paradigm, the software life cycle, requirements analysis, system design, testing and quality assurance. Students will learn the skills of project management, team organisation and risk analysis, mastering the entire software development process and enhancing their overall competence in software development and management.

Course 5: Java Programming

This course provides an in-depth introduction to the Java programming language and platform, including the core features of Java, object-oriented programming, exception handling, I/O operations, and regular expressions. Students will learn about concurrent programming, collection frameworks, generics and GUI building in Java, and master the design and development of Java applications.

Course 6: Principles and Practice of Database Systems

The course is divided into two parts, first introducing the basic concepts of data management such as data models, query languages and database design. Students will then learn about database design and querying, data warehousing and spatial database applications, master the design and implementation of relational databases and be able to apply data management techniques to solve practical problems.

Course 7: Operating Systems Principles

This course explores the structure and services of operating systems, including system calls, process and thread management, memory management, file systems and security. Students will learn the basic principles and structure of operating systems, understand process scheduling, synchronisation and deadlock handling, and master operating system design and implementation.

Course 8: Web Application Development

This course introduces the fundamentals of HTML, CSS and JavaScript, and teaches methods for building and deploying web applications using current web technologies and frameworks. Students will master front-end development techniques and be able to develop and deploy modern web applications, laying the foundation for a career in web development.

On successful completion of this component, students will be able to demonstrate that they can:
①Write and run basic web applications
②Determine how websites and web applications should be built
③Use technologies that support the generation of correct Javascript code on the Node.js platform
④Determine the appropriate platform and modules to use in their applications
⑤Apply appropriate programming concepts to create new web applications.
